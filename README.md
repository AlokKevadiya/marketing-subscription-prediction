# Supervised Learning Assignment – Marketing Subscription Prediction

## Author

Alok Kevadiya

## Dataset

* Name: Marketing and Product Performance Dataset
* Source: https://www.kaggle.com/datasets/imranalishahh/marketing-and-product-performance-dataset

## Objective

The goal of this project is to build and evaluate supervised learning models to predict customer subscription behavior and analyze model performance across multiple research questions.

## Task Type

Classification

## Research Questions

1. What is the baseline performance of a simple model?
2. Which machine learning model performs best?
3. How does preprocessing affect performance?
4. Which features are most important?
5. How do different evaluation metrics impact model selection?
6. How robust are models to data variation?
7. What is the best final model for deployment?

## Methodology

* Data cleaning and preprocessing
* Feature encoding and scaling
* Model training and evaluation
* Comparative analysis across models
* Interpretation of results

## Models Used

* Logistic Regression
* Decision Tree
* Random Forest
* Support Vector Machine
* XGBoost

## Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1-score
* ROC-AUC

## Repository Structure

* notebooks/ → 7 notebooks (one per research question)
* results/figures/ → saved plots (PDF format)
* results/tables/ → saved tables (CSV format)

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Open Jupyter Notebook:
   jupyter notebook

3. Run all notebooks

## Results

Each notebook generates:

* One figure (saved as PDF)
* One table (saved as CSV)

## Notes

* All notebooks are executed with visible outputs.
* Dataset is loaded from Kaggle.
