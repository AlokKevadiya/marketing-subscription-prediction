# Supervised Learning Assignment – Marketing & Product Performance

## Dataset

* Name: Marketing and Product Performance Dataset
* Source: https://www.kaggle.com/datasets/imranalishahh/marketing-and-product-performance-dataset
* Description: Dataset containing marketing metrics, product performance indicators, and customer-related features.

## Objective

To analyze and predict performance using supervised learning models and evaluate them across multiple research questions.

## Research Questions

1. Baseline model performance
2. Model comparison
3. Effect of preprocessing
4. Feature importance and interpretability
5. Sensitivity to evaluation metrics
6. Robustness and generalization
7. Final model recommendation

## Methodology

* Data preprocessing (cleaning, encoding, scaling)
* Model training (Logistic Regression, Decision Tree, Random Forest, XGBoost, SVM)
* Evaluation using multiple metrics
* Comparison and analysis

## Models Used

* Logistic Regression
* Decision Tree
* Random Forest
* XGBoost
* SVM

## Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1-score
* AUC

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Open notebooks:
   jupyter notebook

3. Run all cells

## Results

* Each notebook corresponds to one research question.
* Figures are saved as PDF files.
* Tables are saved as CSV files.
